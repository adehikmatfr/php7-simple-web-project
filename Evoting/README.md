### Evouting

ini adalah aplikasi saya yang di bangun pada tahun 2018, ketika saya duduk di kelas 11 smkn 1 kawali.
Aplikasi ini saya berinama Evouting. aplikasi ini sangat besic menggunakan php native dan database mysql rekomendasi untuk yang sedang belajar pemograman php.
untuk struktur datanya juga menurut saya sudah sedikit rapih ketimbang aplikasi yang lebih awal dari ini. selamat belajar !

## How to use

download aplikasi dengan mengklon repositori git ini , jika anda menggunakan HP anda harus merubah tampilan ke menu desktop. atau jika sudah menginstall git di laptop bisa mengetikan perintah di bawah . dengan PROJECT URL sesuai dengan link repo ini. dan jangan lupa untuk import databse yang di sediakan dan mengkonfigurasinya.

```
$ git clone <PROJECT URL>
```
